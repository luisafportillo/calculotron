Qué es el área
El área es un método para calcular las figuras, es un concepto métrico que permite asignar una medida a la extensión de una superficie, expresada en matemáticas unidades de medida denominadas unidades de superficie. El área es un concepto métrico que requiere que el espacio donde se define o especifique una medida. Para superficies planas, el concepto es más intuitivo. Cualquier superficie plana de lados rectos, por ejemplo un polígono, puede triangularse y se puede calcular su área como suma de las áreas de dichos triángulos. Ocasionalmente se usa el término "área" como sinónimo de superficie, cuando no existe confusión entre el concepto geométrico en sí mismo (superficie) y la magnitud métrica asociada al concepto geométrico (área) y se calcula usando fórmulas de cada figura
